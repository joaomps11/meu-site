Integração de Hardware com Raspberry Pi

Projeto acadêmico voltado ao desenvolvimento de um protótipo funcional de automação, utilizando Raspberry Pi e programação em Python.

O projeto envolveu a integração entre hardware e software, permitindo testes práticos de aplicações em Internet das Coisas (IoT), explorando a comunicação entre dispositivos e o controle de componentes físicos através de código.

Durante o desenvolvimento, foram aplicados conceitos de sistemas embarcados, automação de processos e interação entre sensores e atuadores, proporcionando uma experiência prática no desenvolvimento de soluções integradas.

Essa experiência contribuiu diretamente para o aprimoramento das minhas habilidades em programação, raciocínio lógico e entendimento da relação entre software e hardware.

Tecnologias: Raspberry Pi, Python
