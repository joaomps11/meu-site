Sobre o Projeto

Este projeto foi desenvolvido utilizando Raspberry Pi, um pequeno computador utilizado para experimentos e projetos de hardware e software. O objetivo foi explorar conceitos de programação e automação, permitindo controlar dispositivos e executar tarefas utilizando um sistema embarcado. Esse projeto proporcionou experiência prática com tecnologia e integração entre software e hardware.
